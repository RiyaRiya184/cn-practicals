add router 2911
add two switches below router (left and right)
add a pc to each switch 
connect switches to router
connect pc's to their switches

(configure pc's)
ip address - 192.168.1.2
subnet mask - 255.255.255.0
default gateway-192.168.1.1
(same for other pc)
ip address - 192.168.2.2
subnet mask - 255.255.255.0
default gateway-192.168.2.1

(configure router)
cli
enable
configure terminal
interface gigabitethernet0/0
ip address 192.168.1.1 255.255.255.0
no shutdown 
exit
configure 2nd interface 
interface gigabitethernet0/1
ip address 192.168.2.1 255.255.255.0
no shutdown 
exit

(save configuration) 
end
write

ping from pc ping 192.168.2.2 if reply comes then successful
