# commands 
## 1.ipconfig
ipconfig is a Windows command-line utility used to display and manage the computer’s network configuration details such as IP address, subnet mask, and default gateway.
## 2. ping 
ping google.com <br>
ping is a network command used to check the connectivity between two devices on a network and measure the time taken for data packets to travel from the source to the destination.
## 3.nslookup
nslookup google.com<br>
nslookup is a network command used to find the IP address of a domain name or obtain DNS-related information from a DNS server.
## 4.trace route
tracert google.com<br>
traceroute is a network command used to trace and display the path taken by data packets from the source computer to the destination across a network.
## 5.netstat
netstat is a network command used to display active network connections, listening ports, and network statistics of a computer.
## 6.netstat -a
netstat -a is a command used to display all active network connections and listening ports on a computer.
## 7.
netstat -u
netstat -u is a command used to display all active UDP (User Datagram Protocol) network connections on a computer.
## 8.hostname
hostname is a command used to display the name of the computer or host on a network.
## 9. host
To find the IP address of a domain name.
